## 简介

只是个简单的事件总线，一边post一边监听的那种，不需要过多介绍了，对吧(如果实在想看介绍，了解一下[Bukkit](https://bukkit.org)的事件总线就行了):>


<!-- Features -->
### 功能

- 一个事件post，所有这个事件的监听器都运行!
- 简单使用！

### 环境

需要一个 Gradle 开发环境 (如果你是 Kotlin DSL，那么恭喜你，你的安装方法比 Groovy DSL 的稍微简单一些...)
